# Wireframes - Flowrite 2.0 (Premium SaaS)

## 1. Landing (Welcome Page)
- **Header**: Logo, Nav (Features, Pricing, Contact), Login Button (Primary Outline).
- **Hero**: Large Text ("Your Operation, Simplified"), CTA ("Get Started"), dynamic 3D graphic or glassmorphism effect.
- **Features Section**: 3-Column Grid with icons (Reporting, Dockets, Analytics).
- **Footer**: Simplified Links, Copyright.

## 2. Dashboard Shell (Protected)
- **Sidebar (Left)**:
  - User Profile at bottom.
  - Navigation: Dashboard, Submissions, Dockets, Customers, Items.
  - Brand Logo at top.
- **Top Bar**: Search Bar, Notification Bell, User Avatar.
- **Main Content Area**: Dynamic content based on route.

## 3. Submissions Dashboard (Key Page)
- **Top Controls**: Active Page Title ("Submissions"), Tab Switcher ("List" vs "Calendar").
- **Header Stats**: 3 Cards ("Total Today", "Pending Review", "Revenue").
- **Main List**:
  - Filter Bar: Date Range Picker, Customer Dropdown, Search Input.
  - Table/List View: Rows with Customer Name, Date, Status Badge, Amount.
  - Row Actions: View (Eye), Edit (Pencil), Delete (Trash).
- **Modals**:
  - View Modal: Clean layout, large text for data, image gallery for photos.
  - Edit Modal: Form inputs with validation.

## 4. Docket Generator (Modal/Page)
- **Step 1**: Select Customer (Dropdown with search).
- **Step 2**: Select Date Range.
- **Step 3 (Preview)**: List of matching submissions, Total Amount calculation.
- **Step 4 (Action)**: "Generate PDF" Button (Primary), "Email" Button (Secondary).

## 5. Mobile Layout (Responsive)
- **Sidebar**: Collapses to hamburger menu.
- **Tables**: Collapses to Card List view.
- **Modals**: Full-screen dialogs for easier interaction.
